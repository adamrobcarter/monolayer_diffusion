## Dependencies
* `pip install numpy`
* `pip install scipy`
* `pip install numba`
* `conda install -c conda-forge libmobility` (https://libmobility.readthedocs.io/en/latest/installation.html)
* in a convenient directory, `https://github.com/adamrobcarter/place_colloids.git`, `cd place_colloids`, `pip install -e .`
